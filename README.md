# ideyuta.com [![Build Status][circle-image]][circle-url]

## Architecture

This site use [gatsby](https://github.com/gatsbyjs/gatsby).  
Using the webpack in gatsby to the build and dev-server.

## Development

```vi
npm install
npm start
```

Use `gh-pages`.
